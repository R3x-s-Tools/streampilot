# StreamPilot

**Never Stream Alone.**

StreamPilot is an open-source Creator Intelligence Platform that helps creators improve every stream through thoughtful analytics, actionable insights, reliable reporting, and creator-first tooling.

> Formerly DadR3x Command Center.

## What StreamPilot Does

StreamPilot helps creators understand what happened before, during, and after a stream so every stream can make the next stream better.

Core areas:

- Mission Control
- Analytics
- Reports
- Connectors
- Intelligence
- Optional Producer Notes
- Optional automation integrations

## Project Philosophy

StreamPilot is built around these principles:

- Creator First
- Local First
- AI is Optional
- Security by Design
- Open Source
- Community Supported
- Never Stream Alone
