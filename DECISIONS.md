# Project Decisions

This file captures important project-level decisions in plain language.

For full architectural records, use `docs/adr/`.

## 2026-06-26 — Project Name

Decision: The project name is StreamPilot.

Reason: The new name reflects the long-term vision of a Creator Intelligence Platform rather than a creator-specific dashboard.

## 2026-06-26 — Product Category

Decision: StreamPilot is a Creator Intelligence Platform.

Reason: This describes the platform's broader purpose without limiting it to one streaming service or feature.

## 2026-06-26 — Tagline

Decision: The official tagline is "Never Stream Alone."

Reason: It communicates support, guidance, and confidence for creators.

## 2026-06-26 — AI Optional

Decision: AI is optional.

Reason: StreamPilot must remain valuable without cloud AI, paid AI services, or API keys.

## 2026-06-26 — Reports and Producer

Decision: Reports remain factual. Producer interprets reports.

Reason: Separating facts from coaching keeps StreamPilot trustworthy and extensible.
