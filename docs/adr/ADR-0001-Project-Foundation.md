# ADR-0001 — Project Foundation

## Status

Accepted

## Decision

The project is named StreamPilot, categorized as a Creator Intelligence Platform, uses the tagline "Never Stream Alone," and is licensed under Apache License 2.0.

## Context

The project began as DadR3x Command Center and evolved into a broader creator intelligence platform.

## Consequences

Future work should align with the StreamPilot mission and Foundation Pack.
