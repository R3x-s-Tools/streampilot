# SPS-002 — Product Vision

**Status:** Approved  
**Version:** 1.0

## Product Definition

StreamPilot is a Creator Intelligence Platform.

It helps creators make better decisions before, during, and after every stream through analytics, intelligence, automation, and optional AI.

## Product Pillars

- Intelligence
- Analytics
- Integrations
- Automation
- Trust

## Creator Journey

- Plan
- Create
- Capture
- Analyze
- Improve

## Intelligence Flywheel

Every stream should make the next stream better.

## North Star

Every feature should help creators make better decisions.
