# SPS-009 — Plugin SDK

**Status:** Approved  
**Version:** 1.0

## Purpose

The Plugin SDK allows StreamPilot to be extended without modifying the core platform.

## Principles

- Simple
- Stable
- Secure
- Documented
- Versioned
- Backward compatible when practical

## Boundaries

Plugins must not modify core files, bypass permissions, read secrets directly, or disable security mechanisms.

## Quality

High-quality plugins should be reliable, documented, secure, tested, privacy-respecting, and compatible.
