# SPS-000 — Manifesto

**Status:** Approved  
**Version:** 1.0

## Mission

StreamPilot is a Creator Intelligence Platform that empowers creators to make better decisions before, during, and after every stream.

## Vision

To become the trusted open-source Creator Intelligence Platform.

## Values

- Creator First
- Trust Before Growth
- Intelligence Before AI
- Local First
- Modular by Design
- Security by Default
- Platform Agnostic
- Community Driven

## Non-Goals

StreamPilot will not replace OBS, replace Streamer.bot, require AI, require cloud services, collect telemetry without consent, monetize creator data, lock creators into subscriptions, or automate creativity.

## Promise

The desktop application will provide a complete and useful core experience without requiring a subscription.

Support unlocks gratitude, not functionality.

## Closing

Build the platform we wish had existed when we started streaming.

Never Stream Alone.

Every line of code should leave the platform better than we found it.
