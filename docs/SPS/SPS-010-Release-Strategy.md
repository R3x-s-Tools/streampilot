# SPS-010 — Release Strategy

**Status:** Approved  
**Version:** 1.0

## Principles

- Stability first
- Small releases
- Backward compatibility when practical
- Transparent releases
- Feature readiness

## Channels

- Development
- Preview
- Stable

## Creator Stability Promise

No Stable release should knowingly disrupt a creator's streaming workflow.

## Release Quality Checklist

Feature complete, documented, tested, reviewed, changelog updated, no critical known issues, tagged in Git.
