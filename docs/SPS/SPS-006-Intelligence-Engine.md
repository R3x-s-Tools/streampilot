# SPS-006 — Intelligence Engine

**Status:** Approved  
**Version:** 1.0

## Purpose

The Intelligence Engine transforms raw streaming data into actionable insights.

## Principle

Intelligence is broader than Artificial Intelligence.

## Layers

1. Data Collection
2. Analytics
3. Intelligence
4. Presentation

## Producer

Producer is an optional presentation component that interprets analytics and provides creator-focused coaching.

It enhances but never replaces factual reports.

## Creator Goals

Recommendations should align with the creator's goals rather than assuming viewer count is the only measure of success.

## Guiding Principle

Every stream should make the next stream better.
