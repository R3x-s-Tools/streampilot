# SPS-013 — Glossary

**Status:** Approved  
**Version:** 1.0

## Terms

- Creator: The person using StreamPilot.
- Stream Session: A tracked period of creator activity.
- Event: A factual occurrence.
- Analytics: Measured information.
- Insight: Evidence-based observation.
- Recommendation: Suggested action.
- Report: Structured presentation of session information.
- Producer: Optional coaching presentation component.
- Connector: External integration translator.
- Intelligence Engine: System that turns analytics into insights and recommendations.
- Mission Control: Live operational workspace.
- Plugin: Extension to StreamPilot.
- Platform: The complete StreamPilot ecosystem.
