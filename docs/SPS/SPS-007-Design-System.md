# SPS-007 — Design System

**Status:** Approved  
**Version:** 1.0

## Philosophy

Every interface should answer:

1. Where am I?
2. What can I do here?
3. What should I do next?

## Principles

- Creator First
- Progressive Disclosure
- Consistency
- Feedback
- Accessibility
- Respect the Creator's Focus
- Cross-platform consistency

## Live Experience

While live, StreamPilot should minimize interruptions and avoid competing for attention.

## Guiding Principle

Every interface should help creators focus on creating, not on learning software.
