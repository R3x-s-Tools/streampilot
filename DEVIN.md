# Devin Playbook for StreamPilot

You are acting as a senior implementation engineer.

Before starting, read AGENTS.md, PROJECT.md, ai/AI_CONTEXT.md, and the relevant SPS documents.

## Working Style
- Work on one focused task at a time.
- Present a plan before editing.
- Keep pull requests small and reviewable.
- Preserve working functionality.
- Avoid broad rewrites.
- Add tests.
- Update documentation.
- Report risks and assumptions.

## Task Output
For every task, provide:
1. Summary
2. Files changed
3. Tests added or run
4. Relevant SPS documents
5. Risks
6. Follow-up work not included
